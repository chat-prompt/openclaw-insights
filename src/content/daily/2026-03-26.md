## 📌 주요 논의 토픽 요약

### 1. 보안 vs 실용성 줄다리기
- 걱정: 파일 삭제, 개인정보 유출, 프롬프트 인젝션, 비용 폭탄
- 대응: 권한 최소화, 전용 계정, API 게이트웨이, 갑옷(제한 레이어)
- 문화: "보안 해줘"로 LLM에 위임 → "믿음의 영역"

### 2. 로컬 LLM 실험
- 기대: 비용 절감, 오프라인 가능, 개인정보 보호
- 현실: 속도 느림, 품질 낮음, 하드웨어 비용 vs 클라우드 API 가성비
- 활로: 야간 배치 작업 (임베딩, 요약), 짜잘한 것들

### 3. roach의 메모리 MCP — 기술적 하이라이트
- 중요도 기반 망각 시스템 (헤브의 법칙)
- Auto Dream과 비교, 로컬 최적화 목표
- 커뮤니티 반응: "사람의 기억과 유사하다"

### 4. 덕후방 정체성 논쟁
- 초보자 vs 하드유저 니즈 충돌
- 해결: "가볍게 정보 나누는 방(현재)" vs "하드 유저 전용 방(신설 예정)"
- 슬랙 봇 생태계 실험 예고 (1주일 입주 프로그램)

### 5. 빌더 조쉬 출연 효과
- 유튜브 영상 계기로 신규 유입 급증 (300명 돌파)
- "봇들끼리 협업" 컨셉에 대한 관심 증가

### 6. 초보자 온보딩 문제
- "설치만 하고 뭘 해야 할지 모르겠다" 반복
- 유튜브 영상 추천, 릴리스AI 한 달 결제 추천 (티비 보는 라이언)
- 설치 변수가 많아 트러블슈팅 어려움
